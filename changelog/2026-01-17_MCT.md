# 2026-01-17

## Features
- **Warta Public Viewer**: Implementasi halaman detail warta jemaat (`/warta/[id]`) untuk publik. Halaman ini sekarang menampilkan data real dari database termasuk modul Lagu, Ayat, Tata Ibadah, Pelayan, Pengumuman, Statistik, Keuangan, dll.
- **Smart Lyric Fetching**: Modul Lagu dan Tata Ibadah pada view publik sekarang otomatis mengambil lirik berdasarkan ID lagu yang terhubung.
- **User Experience**: Penambahan kontrol ukuran font (Text Sizer) pada halaman publik Warta untuk aksesibilitas yang lebih baik.
- **Editor Enhancements**:
  - Implementasi **Drag and Drop** (`dnd-kit`) untuk mengatur urutan modul di halaman Admin Warta.
  - Penambahan fitur **Collapse All / Expand All** untuk mempermudah navigasi saat mengedit banyak modul.
  - Perbaikan UI dropdown "Tambah Modul".

## API Improvements
- **Song Search**: Update endpoint `/api/songs` untuk menangani query spesifik kategori + nomor (contoh: "KJ 123", "BE 456") secara presisi, memisahkan logika pencarian teks biasa dan pencarian kode lagu.

## Data Maintenance
- **Song Data Cleanup**: Penambahan scripts (`scripts/cleanup-songs-be.ts`) dan SQL queries untuk membersihkan judul lagu (menghapus suffix "- BatakPedia") dan menormalisasi nomor lagu pada kategori BE.
